**Tools: C++, Python, ROS, Arduino, PID controller.**

A mini version of The ABU Asia-Pacific Robot Contest, each game is between two teams, the playing field is symmetrical and robots from both teams start at the same condition.

Specifically, a regular game may lasts from 3 to 5 minutes for both teams to control their robots and collect items as much as possible. Each team has a mannual robot and an automatic robot. The data from jury servers is processed and fed directly to the automatic one in order to perform certain tasks.
